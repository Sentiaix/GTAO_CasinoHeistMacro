카지노 습격 지문 매크로 V1.2.1

-----------------------------------------------------------------------------------

이 프로그램은 학습 목적으로 만들어졌으며 pyautogui의 모듈 학습 과정 중 만든 프로그램입니다.

GTA 5에서 등장하는 카지노 습격의 지문 해킹을 주제로 만들었습니다.
지문 해킹은 우측에 큰 지문이 나오고, 좌측에는 올바른 지문 조각 4개와 잘못된 지문 조각 4개,
총 8개의 지문 조각이 나오며, 올바른 지문 조각 4개를 맞춘 후 검증을 하는 시스템입니다.

1920 x 1080 (FHD)해상도에서 캡쳐한 지문 조각 이미지를 가우스 블러 로직을 이용하여
화면에 보이는 이미지와 대조, 자체적으로 점수를 매겨 pyautogui와 cv(computer vision)를
통하여 하드웨어 수준의 입력을 시행합니다.

-----------------------------------------------------------------------------------

# 읽어주세요 (README) #

첫 실행 이전에는 "config.ini"파일이 없습니다. 첫 실행 이후 자동 생성됩니다.
만약 프로그램이 오작동한다면 "config.ini"파일을 찾아서 세팅값을 조정해주세요.
"config.ini" 파일에서 HOTKEYS에 키 이름을 쓸 때는, 대소문자 구별을 하지 않습니다.
다중 키 입력을 지원하기는 하나, 오류가 많아 가급적 펑션키(F1~F12) 사용을 추천드립니다.
ESC키 사용도 권장드릴 수 있으나 GTA5의 '일시 정지 메뉴'키의 기본값이므로 추천드리지 않습니다.

1920x1080 해상도에서 제작되어서 프로그램은 1920x1080 해상도에 최적화 되어있습니다.
