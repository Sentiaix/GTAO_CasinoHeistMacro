# GTA5 Casino Heist Macro (실험용)
GTA5 지문 해킹을 자동화하는 파이썬 프로그램입니다.

## 단축키 안내
- **F4**: 프로그램 종료
- **F5**: 해킹 시작 (지문 화면에서 사용)
- **F6**: 프로그램 재시작 (해상도 변경 시 사용)
- **F7**: 시스템 활성화 상태 확인 (부저음)

## 다운로드
- 옆 Release 버튼(PC기준)을 눌러 `.exe`를 다운받거나,
- [이 링크](https://github.com/Sentiaix/GTAO5_fingerMacro/releases/tag/macro)를 클릭하여 아래로 스크롤한 후, .exe를 다운받습니다.
- **단, 가장 높은 버전의 파일을 다운로드 하시는걸 권장합니다. (예: 1.0.1(x) 1.2.2(o))**

## 주의사항
- 실행에 오류가 있다면 **관리자 권한**으로 실행하는걸 권장합니다.
- 본 프로그램은 교육 및 연구 목적으로만 제공됩니다.
- **컴퓨터 사양이 낮거나, GTA5 실행 자체로 컴퓨터에 부하가 걸린다면 프로그램 실행에 지연이 될 수 있습니다.**

## 추가 안내사항
- 이 프로그램은 `1920x1080`해상도에 맞춰 제작되었습니다.
- 해상도 적응 기술을 추가하였지만, 인식률이 낮을수 있습니다.
- 인식이 안된다면 `1920x1080`해상도로 변경 후, 창모드로 이용하시길 권장드립니다.
- 또한 모니터가 여러개인 경우, 가장 좌측 모니터에 게임을 두고 실행하시기를 권장드립니다. (추후 개선할 예정)
